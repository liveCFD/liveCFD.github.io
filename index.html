liveCFD

liveCFD provides a complete Open Source CFD suite ready to be loaded into the Slackware® Live OS.

The CFD environment maintains compatibility with the standards defined in blueCFD®-Core for OpenFOAM®.org and ParaView®, to which it adds Blender® as a preprocessor. So the heroes we are especially devoted to are: tkeskita (Tuomo Keskitalo) for his GUI addon, wyldckat (Bruno Santos) for the de-facto CFD standardization, and AlienBOB (Eric Hameleers) for the excelent live Linux® OS. Surely in turn they will be devoted to other great heroes and benevolent dictators.

How to configure the Module:

0. Chose your preferite liveslak ISO variant and boot with it on any x86_64 machine; optionally chose some other nice bonus functionality to add on, e.g. NVIDIA® drivers, etc.

1. Use the "iso2usb.sh" script to transfer the ISO content to a USB disk;

2. Download the Module: liveCFD-Module-2020-1-current-x86_64

liveCFD-Documentation-2020-1-current

3. Copy the Module into the "liveslak/addons" subdirectory of the USB disk; optionally copy the further addons in the same subdirectory.

Enjoy!
